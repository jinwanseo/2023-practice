# Get Started Server

## Step 1: Install

- install npm modules

```
    npm install
```

- install postgreSQL

## Step2: Set Server

- create postgreSQL Database Table

```
    CREATE DATABASE standby;
```

- make .env file in root folder

- include database url, save

## Step3 : Start Server

```
    npm start
```
